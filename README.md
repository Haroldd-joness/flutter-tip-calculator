# tip_calculator

A flutter Tip Calculator. 

## Getting Started

This project is an easy tip calculator made with flutter. The application calculates 
tips between three percentage options (15%, 20% and 50%).
##Packages used 
lottie - for the splash screen animation 
google fonts
